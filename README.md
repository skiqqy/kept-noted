# kept-noted
The note daemon (noted) project. This is the client which interacts with the
[server](https://github.com/skiqqy/kept-folder).

## Requirements

* [dmenu](https://tools.suckless.org/dmenu/)
* [jq](https://stedolan.github.io/jq/)
* bash

## Installing
`$ sudo make install`

## Usage
`$ noted`

## Uninstalling
`$ sudo make uninstall`
